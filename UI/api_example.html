Endpoints

Trending Topics

Retrieve the current trending topics across multiple platforms.

GET

Parameters

No parameters required for this endpoint.

Example Response

{
  "data": [
    {
      "topic": "#AIForGood",
      "platform": "Twitter",
      "volume": "1.2M"
    },
    {
      "topic": "Sustainable Tech",
      "platform": "Google",
      "volume": "850k"
    }
  ]
}

Search Trends

Search for trends related to a specific keyword.

GET

Parameters

q string, required

The keyword to search for.

platform string

Filter by platform (e.g., 'twitter', 'google').

Example Response

{
  "data": [
    {
      "topic": "Serverless Architecture",
      "platform": "Google",
      "relevance": "95%"
    },
    {
      "topic": "#Serverless",
      "platform": "Twitter",
      "relevance": "89%"
    }
  ]
}
